# 03 — Design system

На основе `analysis/01_reverse_engineering.md` и `analysis/02_component_catalog.md`

реконструируй вероятную design system эталона (как основу языка LifeOS).

## Включить

- Typography
- Color Tokens
- Spacing Tokens
- Radius Tokens
- Elevation
- Motion
- Icons
- Grid
- Layout
- Breakpoints (mobile-first; desktop не обязателен в V1)
- Component Rules
- Naming conventions
- Design principles

## Правила

- Если возможны разные интерпретации — перечисли и укажи степень достоверности.
- Никогда не создавай иллюзию точности там, где её нет.
- Задавай вопросы, когда доказательств недостаточно.
