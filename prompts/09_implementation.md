## Стек и платформа

- Vue 3 + Vite (+ TypeScript по `TECH.md`)
- PWA: Add to Home Screen на iOS
- **Полноценный offline обязателен** с первой рабочей сборки
- Persistence строго по утверждённой модели данных

## Правила

1. Ничего не перепроектировать. Не «улучшать UX». Не добавлять свои продуктовые идеи.
2. Если реализация требует предположения — стоп и вопрос.
3. Никогда не менять дизайн молча.
4. Внедрять **по одному экрану** (или согласованному вертикальному срезу) за раз.
5. После каждого экрана — короткое self-review и ждать одобрения.

## Self-review после экрана

Искать:

- несоответствия утверждённому дизайну
- дублированный код
- ненужные абстракции
- архитектурные запахи
- проблемы с maintainability
- регрессии offline / PWA

Сначала зафиксировать fidelity экрана. Рефакторинг без смены UX — только после апрува экрана, отдельным мини-шагом.
