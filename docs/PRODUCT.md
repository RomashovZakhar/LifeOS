# Product Scope

## Версия 1 (обязательно всё сразу)

1. **Habits** — отслеживание привычек (корень приложения, сетка эталона)
2. **Workouts** — полноценный модуль тренировок (вход через колонку-портал на Habits)
3. **Checklist** — именованные чеклисты с прогрессом в %  
   *(в ранних формулировках — Morning / утренний ритуал; продуктово не ограничено утром)*

Три возможности — не опциональные фазы разработки. Все входят в первый релиз.  
Колонки-порталы пользователь может не создавать — модули при этом остаются в продукте.

Навигация и границы: `analysis/04_information_architecture.md`.  
Данные: `analysis/04b_data_model.md`.

---

## Ограничения по продукту

Всё должно выглядеть как одно приложение.

Ни один модуль не должен вводить совершенно другую модель взаимодействия.

На каждом новом экране по возможности использовать существующие шаблоны проектирования (из эталона и уже принятых решений LifeOS).

## Платформа

- Vue PWA
- Установка на iPhone через Add to Home Screen
- Полноценный offline (см. `TECH.md`)

---

## Производительность

- Запуск приложения < 1 секунда (ощущаемый cold start на устройстве)
- Типичные действия < 15 секунд end-to-end
- Большинство действий ≤ 3 нажатия
